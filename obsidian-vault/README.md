# Obsidian Vault — JARVIS Cloud

This directory will be git-synced with the JARVIS Obsidian vault.

## Planned Structure

```
obsidian-vault/
├── Agents/            — Agent profiles, session logs, task history
├── Projects/          — BeldiTalk, Trading, Infinity Mirror Maroc
├── Skills/            — Hermes skill definitions and documentation
├── Daily/             — Daily notes and reflections
├── Templates/         — Note templates
└── .obsidian/         — Obsidian config (excluded from git if sensitive)
```

## Setup

1. Clone this repo to your local machine
2. Open Obsidian → "Open folder as vault" → select this directory
3. Enable git plugin or use external git sync
